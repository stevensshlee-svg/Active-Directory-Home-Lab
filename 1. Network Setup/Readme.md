# Overview
This section documents the initial lab infrastructure configuration prior to deploying Active Directory

## Configurations Performed
* Installed Windows Server 2019
* Configured a static IP address on the server
* Installed AD DS role
* Promoted server to Domain Controller
* Installed and configured DNS
* Configured the server's DNS IP address as itself
* Verified SRV records
* Joined Windows 10 client machine to domain

## Validation Steps
* Verified DNS resolution using nslookup
* Verfied domain join via system properties
* Confirmed AD connectivity via login test
